# Django Portfolio Projects

This repository contains **two complete, separate, and fully functional Django projects** placed in their own folders. Both are responsive, clean, and deployed live on Render (free tier).

## 1. BMI Calculator (`/bmi-calculator`)

A fast, beautiful Body Mass Index calculator that works instantly — no login required.

**Live Demo – BMI Calculator**  
https://bmi-yourusername.onrender.com  
[![BMI Live](https://img.shields.io/badge/BMI_Calculator-Live-brightgreen?style=for-the-badge&logo=render)](https://bmi-yourusername.onrender.com)

**Features**
- Supports **Metric (kg/cm)** and **Imperial (lbs/ft-in)** units
- Real-time calculation using the standard BMI formula
- Color-coded results: Underweight, Normal, Overweight, Obese
- Mobile-first responsive design with Bootstrap 5
- Single-page application — super fast and simple

**Perfect for learning**
- Django forms & GET/POST handling
- Template logic and conditional rendering
- Clean, reusable Bootstrap UI

Live → https://bmi-yourusername.onrender.com

## 2. MyBlog (`/myblog`)

A complete personal blogging platform with a modern, elegant look.

**Live Demo – MyBlog**  
https://myblog-yourusername.onrender.com  
[![Blog Live](https://img.shields.io/badge/MyBlog-Live-0066CC?style=for-the-badge&logo=render)]([https://myblog-yourusername.onrender.com](https://django-projects-4lgq.onrender.com/  ))

**Features**
- Create, edit, and delete posts (via Django admin or custom views)
- Automatic post dates and readable formatting
- Responsive layout that looks great on phones, tablets, and desktops
- Clean URLs (e.g., `/post/my-new-article/`)
- Easy to extend (comments, categories, tags, etc.)

**Perfect for learning**
- Models, migrations, and admin customization
- Class-based views and template inheritance
- Static files and Whitenoise in production

Live →  https://django-projects-4lgq.onrender.com/ 
Admin → https://myblog-yourusername.onrender.com/admin

### Tech Stack (Both Projects)
- Django 5.x
- Bootstrap 5
- SQLite (zero setup)
- Whitenoise (static files in production)
- Deployed on Render.com
